# Y
Y is a language that is written in `c` (for now, soon to be selfhosted (somehow))  
It is made to make the program of [yes](https://github.com/mubaris/yes) easier.  

# Usage
the whole syntax of y looks like this
```y
y
```
and to compile

```bash
y y.y
```

To run the program
```bash
./y 
```

this would result in a output of   

```
y
y
y
y
...
```

perhaps you want something else, maybe a different string. that would mean you need to pass it as a argument  

```bash
./y y.y 7a8e9
```

which would result in   
```
7a8e9  
7a8e9  
7a8e9  
7a8e9  
7a8e9  
...
```
